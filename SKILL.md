# vault-bulk-reorganization

Structured protocol for bulk vault operations — auditing and reorganising existing directories, fixing frontmatter across multiple notes, and deciding the fate of legacy files. Use whenever a task involves reviewing/restructuring a vault directory with 3+ files, fixing frontmatter across multiple notes, triaging 'decide fate'/'clean up'/'reorganise' requests, or archiving/moving/deleting vault files in bulk.
